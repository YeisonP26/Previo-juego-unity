<img width="1365" height="767" alt="image" src="https://github.com/user-attachments/assets/58376a9f-5921-4563-ad3c-d8e7cfe39df8" />📌 Descripción

Este proyecto es una aplicación desarrollada en Unity, que contiene una escena interactiva con diferentes elementos como prefabs, scripts y configuraciones gráficas. Está diseñado como parte de un trabajo académico o práctica de desarrollo de videojuegos.

El proyecto incluye mecánicas básicas de interacción y utiliza el sistema de entrada moderno de Unity.

⚙️ Tecnologías utilizadas
Unity Engine
C# (scripts)
Input System de Unity
Universal Render Pipeline (URP)
📁 Estructura del proyecto
Assets/
Scenes/ → Contiene las escenas del proyecto
Scripts/ → Código en C# para la lógica del juego
Prefabs/ → Objetos reutilizables
Sprites/ → Recursos visuales
Settings/ → Configuraciones del proyecto
ProjectSettings/ → Configuración general de Unity
.vscode/ → Configuración para desarrollo en Visual Studio Code
🚀 Cómo ejecutar el proyecto
Instala Unity Hub
Abre Unity y selecciona:
"Open Project"
Busca la carpeta del proyecto
Abre la escena principal:
SampleScene
Presiona el botón ▶️ (Play) para ejecutar
🎮 Funcionalidades principales
Interacción con objetos en escena
Uso de prefabs reutilizables
Sistema de entrada configurado
Renderizado con URP
📷 Vista general

El proyecto contiene una escena base (SampleScene) donde se pueden visualizar los elementos principales del entorno.
